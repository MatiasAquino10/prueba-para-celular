# Guía de Minecraft para Principiantes
 
## Descripción
 
Este proyecto es una guía interactiva sobre Minecraft orientada a jugadores nuevos. Explica cómo sobrevivir los primeros momentos del juego, qué materiales recolectar, cómo craftear objetos básicos y a qué enemigos prestarle atención.
 
## Contenido
 
- Menú principal con navegación interna
- Sección de primeros pasos con lista de acciones ordenadas
- Tabla de materiales básicos del juego
- Guía de crafteo con objetos esenciales
- Información sobre mobs y enemigos
- Video tutorial embebido de YouTube
- Formulario de contacto
